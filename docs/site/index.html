KSL-BASH-LIB Documentation

Documentation for KSL-BASH-LIB version 0.0.1.

KSL-BASH-LIB provides a collection of bash functions to simplify writing bash scripts. It is primarily focused on supporting the writing of shell scripts as tools, and less so on interactive shell enhancements.

See the source code, documentation source, and build automation at the KSL Bash Library Project on GitHub.
